# 🎴 NIVEL ARENA DECK SYSTEM

![Version](https://img.shields.io/badge/version-1.0.0-green)

專為 **NIVEL ARENA TCG** 玩家打造的網頁版牌組構築工具。提供快速、直覺的牌組建立與查看。

> **開發者備註**：本人非程式專業，此工具完全依靠 AI 來製作，也將此專案開源，如有任何問題與建議歡迎聯繫我。

---

## ✨ 核心特性

* **📱 全方位響應式設計**：針對 PC 與行動端優化。PC 端支援拖放 (Drag & Drop)；行動端支援長按查看詳情與流暢的彈窗操作。
* **🔍 過濾系統**：可依據 IP、系列、屬性、費用 (Cost)、Trigger 等條件精確篩選數百張卡牌。
* **📊 即時數據分析**：自動生成費用分佈長條圖與卡牌種類圓餅圖，讓牌組節奏一目了然。
* **🚨 智能規則校驗**：
    * 自動偵測 **Leader** 是否配置。
    * 檢查 **Trigger** 數量（上限 8 張）。
    * 檢查單卡重複張數（上限 3 張）。
    * 校驗 **IP 一致性** 與 **領袖誓約**。
* **📸 牌組快照**：可將牌組輸出成圖片，除了方便查看與交流外，也可透過讀取功能，讀取該圖片的牌組資料。

---

## 🛠️ 功能指南

### 1. 牌組編輯
* **新增**：從卡牌資料庫拖曳卡片至卡牌區中加入。行動端則透過 + 號按鈕來查看卡牌資料庫並進行加入。
* **調整**：懸停於卡牌區中的卡片上可增加/減少張數。行動端可在卡牌資料庫進行相關操作，或者點擊卡牌區的卡片進入卡牌詳情來操作。
* **詳情**：在任意區域點擊卡牌可在卡牌詳情區中查看。行動端可在卡牌資料庫長按卡片進入，或者點擊卡牌區的卡片進入。

### 2. 存檔與分享
* **儲存牌組**：導出 `.csv` 檔案或將牌組透過牌組列表「Deck List」存入瀏覽器快取。
* **讀取牌組**：支援讀取 `.csv` 或直接上傳曾導出的「牌組快照」圖片。
* **牌組快照**：生成一張包含 Leader 與所有卡牌配置的高解析度分享圖。

### 3. 預設牌組
* 牌組列表內建 **ST** 系列起始牌組，點擊選單即可一鍵讀取。
